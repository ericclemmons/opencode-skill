# opencode-skill

An [OpenCode](https://opencode.ai) skill that provides reference documentation for the OpenCode CLI, Server, SDK, and Formatters.

## Usage

Install as a skill in your OpenCode project:

```bash
opencode skill add ericclemmons/opencode-skill
```

## What's Included

- **[CLI](./references/cli.md)** – Commands, flags, and environment variables
- **[Server](./references/server.md)** – Headless HTTP API server
- **[SDK](./references/sdk.md)** – Type-safe JS/TS client
- **[Formatters](./references/formatters.md)** – Auto-formatting configuration
